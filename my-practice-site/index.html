Выполнил: Никита Якутин

Десктопное приложение 1С

Полнофункциональное рабочее место для администратора. Реализованы справочники, документы, регистры и отчеты для полноценного ведения учебного расписания. Интерфейс адаптирован для работы на больших экранах.

Главное меню администратора
Форма документа "РасписаниеЗанятий"
Отчет "Расписание на неделю"

Мобильное приложение (Android)

Легкий мобильный клиент для преподавателей и студентов. Предоставляет быстрый доступ к актуальному расписанию, работает в офлайн-режиме и имеет простой интерфейс, адаптированный для смартфонов.
